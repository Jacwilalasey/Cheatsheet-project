Margins & Headers

Margins

All the margin properties can have the following values: auto - the browser calculates the margin, length - specifies a margin in px, pt, cm, etc, % - specifies a margin in % of the width of the containing element, inherit - specifies that the margin should be inherited from the parent element. Negative values are allowed.

Tag Name Description
margin-top margin top Adds margin to the top of an element, outside of any defined borders.
margin-right margin right Adds margin to the top of an element, outside of any defined borders.
margin-bottom margin bottom Adds margin to the top of an element, outside of any defined borders.
margin-left margin left Adds margin to the top of an element, outside of any defined borders.
margin margin (shorthand) Adds margin all sides, or 1, 2, 3 sides (e.g. margin: 25px 50px 75px 100px)

Padding

All the padding properties can have the following values: length - specifies a padding in px, pt, cm, etc. % - specifies a padding in % of the width of the containing element, inherit - specifies that the padding should be inherited from the parent element Negative values are not allowed.

Tag Name Description
padding-top padding top Generates space around the top of an element's content, inside of any defined borders.
padding-right padding right Generates space around the top of an element's content, inside of any defined borders.
padding-bottom padding bottom Generates space around the top of an element's content, inside of any defined borders.
padding-left padding left Generates space around the top of an element's content, inside of any defined borders.
padding padding shorthand Generates space around 1, 2, 3 or all sides.
